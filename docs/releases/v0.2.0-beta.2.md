## ⏱️ What is c7pc?

c7pc is a desktop port of Chal7777 — a timing game where you try to stop the timer at exactly 7.777 seconds. The countdown display fades out along the way, so you have to trust your inner clock.

## ✨ New Features

- Landing in the top 3 is now a real celebration 🎉 — confetti bursts up from the bottom of the screen and a big おめでとう！ overlay (gold / silver / bronze to match your rank) shows before the ranking table appears
- The ranking table now shows a trailing ⋮ when you are in the top 10 but more results are hidden below

## 🐛 Bug Fixes

- Fixed the minimum window size, which had width and height swapped — it is now 900 x 800 (width x height)

> [!NOTE]
> 🍎 **macOS users**:
>
> - The app is not code-signed. After moving it to the Applications folder, open Terminal and run the following command:
>   ```sh
>   xattr -r -d com.apple.quarantine /Applications/c7pc.app
>   ```
> - On first launch, Gatekeeper may block it.
>
> 🪟 **Windows users**:
>
> - If SmartScreen shows a warning, click "More info" → "Run anyway".
