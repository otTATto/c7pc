## ⏱️ What is c7pc?

c7pc is a desktop port of Chal7777 — a timing game where you try to stop the timer at exactly 7.777 seconds. The countdown display fades out along the way, so you have to trust your inner clock.

## ✨ New Features

- First stable release 🎉 (same content as v0.1.0-beta.2)
- Full port of the original Chal7777 game flow as a native desktop app: Top → Play → Result
- Launches in true borderless fullscreen (the taskbar is hidden too), with a fullscreen / windowed toggle at the top-right of the title screen
- Timer measurement runs on the Rust backend — the official time is recorded the moment you press stop, independent of the UI refresh interval
- Space key and mouse click support on every screen, just like the web version

> [!NOTE]
> 🍎 **macOS users**:
>
> - The app is not code-signed. After moving it to the Applications folder, open Terminal and run the following command:
>   ```sh
>   sudo xattr -d com.apple.quarantine /Applications/c7pc.app
>   ```
> - On first launch, Gatekeeper may block it.
>
> 🪟 **Windows users**:
>
> - If SmartScreen shows a warning, click "More info" → "Run anyway".
